Mi Tienda Flask - Versión 1.0
----------------------------------

Desarrollado por: Proyectemos
Fecha de publicación: Julio de 2025

Descripción:
Mi Tienda Flask es una aplicación de gestión para ventas, usuarios, cotizaciones y facturación. Está diseñada para facilitar las operaciones de negocios pequeños y medianos, permitiendo gestionar clientes, productos y servicios de manera eficiente.

Instalación:
1. Ejecute el instalador setup_mitienda.exe como administrador.
2. Acceda a la aplicación mediante el acceso directo creado en el escritorio o en el inicio automático.
3. También puede ingresar a la aplicación desde su navegador en: http://www.mitienda.com

Requisitos:
- No se necesita Python ni ningún software adicional instalado.
- El instalador configura todo automáticamente.
